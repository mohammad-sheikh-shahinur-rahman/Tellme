# TellMe - Private & Encrypted Messaging

TellMe is a secure, private messaging platform designed for sending heart-felt letters and messages with end-to-end encryption. Express yourself freely through text, images, and voice recordings, knowing that your privacy is protected.

## Features

- **End-to-End Encryption**: Every letter is encrypted on the client-side before being sent, ensuring only the intended recipient can read it.
- **Customizable Templates**: Choose from various beautiful design templates to match the mood of your message.
- **Multi-Media Support**: Attach photos or voice recordings to add a personal touch to your letters.
- **Dashboard**: A clean, modern dashboard to manage your received letters and reply to them securely.
- **Unique Usernames**: Each user gets a unique profile link (e.g., tellme.app/username) for receiving messages.

## Security

- **AES-256 Encryption**: Used for all message contents.
- **Firebase Authentication**: Secure user login and profile management.
- **Firestore Security Rules**: Strict access control ensuring users can only access their own data.
